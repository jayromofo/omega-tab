# Keyboard Shortcuts

OmegaTab offers a variety of keyboard shortcuts to enhance your productivity and navigation experience. This guide covers all the keyboard shortcuts available in the application.

## Quick Reference

| Action | Shortcut |
|--------|----------|
| Open Command Palette | <kbd>`Ctrl`</kbd> + <kbd>`K`</kbd> |
| Show Keyboard Shortcuts | <kbd>`?`</kbd> (<kbd>`Shift`</kbd> + <kbd>`/`</kbd>) |
| Add New Link | <kbd>`Ctrl`</kbd> + <kbd>`Alt`</kbd> + <kbd>`N`</kbd> |
| Cycle Search Engines | <kbd>`Ctrl`</kbd> + <kbd>`↑`</kbd> / <kbd>`↓`</kbd> |
| Open Link in Column 1 | <kbd>`Ctrl`</kbd> + <kbd>`Number`</kbd> (1-9) |
| Open Link in Column 2 | <kbd>`Ctrl`</kbd> + <kbd>`Alt`</kbd> + <kbd>`Number`</kbd> (1-9)|

## Navigation Shortcuts

### Command Palette

The Command Palette gives you keyboard access to all your links and application commands.

- **Open Command Palette**: <kbd>`Ctrl`</kbd> + <kbd>`K`</kbd>
- **Navigate Options**: <kbd>`↑`</kbd> / <kbd>`↓`</kbd> arrow keys
- **Select Option**: <kbd>`Enter`</kbd>
- **Close Palette**: <kbd>`Esc`</kbd>

![Command Palette](/copy/command_palette.png)

### Help Menu

- **Show Keyboard Shortcuts**: <kbd>`?`</kbd> (<kbd>`Shift`</kbd> + <kbd>`/`</kbd>)

## Search Shortcuts

### Search Bar

- **Cycle Through Search Engines**: <kbd>`Ctrl`</kbd> + <kbd>`↑`</kbd> / <kbd>`↓`</kbd> arrow keys
- **Navigate Search Results**: <kbd>`↑`</kbd> / <kbd>`↓`</kbd> arrow keys
- **Select Result**: <kbd>`Enter`</kbd>
- **Add New Line in Search Query**: <kbd>`Shift`</kbd> + <kbd>`Enter`</kbd>

## Link Management

- **Add New Link**: <kbd>`Ctrl`</kbd> + <kbd>`Alt`</kbd> + <kbd>`N`</kbd>
- **Open Link (Column 1)**: <kbd>`Ctrl`</kbd> + <kbd>`Number`</kbd> (1-9)
- **Open Link (Column 2)**: <kbd>`Ctrl`</kbd> + <kbd>`Alt`</kbd> + <kbd>`Number`</kbd> (1-9)

::: info Column Shortcuts
The first two columns in your dashboard have shortcuts reserved for them. Additional columns will not have a unique shortcut mapped to them.

OmegaTab is designed with a 1 or 2 column layout in mind, and we make 18 links added to your first 1 or 2 columns easy and fast to launch via shortcuts. You can add as many columns as you want, and OmegaTab supports displaying them, but shortcuts are only available for the first two columns.
:::


## Search Engine Shortcuts

You can quickly switch between search engines using keyboard shortcuts:

1. Press <kbd>`Ctrl`</kbd> + <kbd>`↑`</kbd> to move to the previous search engine
2. Press <kbd>`Ctrl`</kbd> + <kbd>`↓`</kbd> to move to the next search engine

Alternatively, you can use the Command Palette:
1. Press <kbd>`Ctrl`</kbd> + <kbd>`K`</kbd> to open the Command Palette
2. Type the name of a search engine
3. Press <kbd>`Enter`</kbd> to select it

## Dialog Navigation

When a dialog box is open (for example, the "Add New Link" dialog):

- **Navigate Fields**: <kbd>`Tab`</kbd>
- **Submit Form**: <kbd>`Enter`</kbd>
- **Close Dialog**: <kbd>`Esc`</kbd>
